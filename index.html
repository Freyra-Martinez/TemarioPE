Temario Probabilidad y Estadistica

Tema 1. Estadistica descriptiva

Subtemas

  • 1.1 Conceptos básicos de estadistica
    • Teoría de decisión:

      • La teoría de decisión se basa en la probabilidad y en la utilidad, y utiliza modelos matemáticos para ayudar al individuo o al grupo a tomar decisiones informadas y racionales

    • Población:

      • La población constituye del total de elementos o individuos que tienen en común una característica e integran un conjunto de sujetos de investigación.

    • Muestra aleatoria:

      • Se trata de obtener una muestra de una población, fundamentada en una probabilidad preferencia de elementos.

    • Parámetros aleatorios:

      • Define valores numéricos a los probables resultados de un ensayo probabilistico para poder mostrar cada ecenario.

  • 1.2 Descripción de datos
    • Datos agrupados:

      • Los datos agrupados son aquellos que están organizados en una función , presentando una frecuencia o grupo.

    • Datos no agrupados:

      • Los datos que no han sido sujetos a ninguna agrupación o categorización después de su recopilación se denominan datos no agrupados. En términos simples, esto significa que los datos no se han dividido en grupos específicos según ningún criterio particular.

    • Frecuencia de clase:

      • La frecuencia de clase es el recuento o porcentaje de puntos de datos que se encuentran dentro de un intervalo de clase específico en una distribución estadística combinada o un histograma derivado de él.

    • Frecuencia relativa:

      • La frecuencia relativa es una medida estadística calculada como el cociente de la frecuencia absoluta de algún valor (fi) dentro de los valores totales de la población/muestra (N).

    • Punto medio:

      • La clase promedio (o símbolo de clase) es un indicador específico de la posición de la base (clase) en la tabla de distribución de frecuencias;

    • Límites

      • El límite nos indica el valor del rendimiento a medida que el valor de entrada se acerca a un número determinado.

  • 1.3 Medidas de tendencia central
    • Media aritmética:

      • La media aritmética es el promedio que da igual peso a todos los valores .

    • Geométrica:

      • Se define como el índice base n del producto de n elementos. La media geométrica se utiliza a menudo para calcular el crecimiento medio de algunas series a lo largo del tiempo.

    • Ponderada:

      • Es una media de centralización que da una importancia distinta a cada uno de los valores sobre los que se calcula la media

    • Mediana:

      • La mediana es un número (estadístico) que divide una lista de valores en dos partes iguales. Esto significa que la mitad de los valores son menores que la mediana y la otra mitad son mayores.

    • Moda:

      • La moda estadística es aquel valor que, dentro de un conjunto de datos, se repite el mayor número de veces.

    • Medidas de dispersión:

      • Las medidas de dispersión dan información sobre el grado de variabilidad de una variable estadística. Esto se consigue a través de diferentes fórmulas que nos dan un valor numérico para valorar el grado de dispersión.

    • Varianza:

      • La varianza es una medida de dispersión que representa la variabilidad de una serie de datos con respecto a su media.

    • Desviación estándar:

      • La desviación estándar es una medida de extensión o variabilidad en la estadística descriptiva. Se utiliza para calcular la variación o dispersión en la que los puntos de datos individuales difieren de la media.

    • Desviación media:

      • La desviación media es una medida importante de dispersión que nos permite entender cuánto varían los datos de un conjunto promedio.

    • Desviación mediana:

      • La desviación mediana es una medida de dispersión estadística que indica cuánto tienden a desviarse los valores de un conjunto de datos respecto a la mediana.

    • Rango:

      • El rango es un valor numérico que indica la diferencia entre el valor máximo y el mínimo de una serie de datos o muestra estadística.

  • 1.4 Parámetros para datos agrupados.

    • Los parámetros principales para datos agrupados son:

    • Tamaño del grupo (n):

      • Este se refiere al número de elementos que conforman cada grupo o categoría.

    • Media del grupo (x̄):

      • Es el promedio aritmético de los valores dentro de cada grupo. Varianza del grupo (s²): Mide la dispersión de los valores dentro de cada grupo con respecto a su media.

    • Varianza del grupo (s²):

      • Mide la dispersión de los valores dentro de cada grupo con respecto a su media.

    • Desviación estándar del grupo (s):

      • Es la raíz cuadrada de la varianza. Indica cuánto se desvían los valores del grupo respecto a la media.

    • Frecuencia del grupo (f):

      • Es el número de elementos o la cantidad de veces que se repite cada grupo o categoría.

  • 1.5 Distribución de frecuencias.

    • Se refiere a la organización y presentación de datos crudos en forma de tablas, gráficos o diagramas.

  • 1.6 Técnicas de agrupación de datos.

    • Las técnicas de agrupación de datos son métodos utilizados en estadística y análisis de datos para organizar conjuntos de datos grandes o complejos en grupos más manejables, con el fin de facilitar su análisis, interpretación y presentación.

  • 1.7 Técnicas de muestreo.

    • son métodos sistemáticos utilizados en estadística y en la investigación para seleccionar una muestra representativa de una población más grande.

    • Tema 2. Fundamentos de la Teoría de Probabilidad

    Subtemas

  • 2.1 Técnicas de Conteo
      • Se refiere a los metodos o formulas que nos ayuda a determinar el total de resultados posibles en un evento o experimento.
    • 2.1.1 Principio aditivo.

      • Se conoce como el numero de ocasiones que existe un evento en un espacio muestral

    • 2.1.2 Principio multiplicativo.

      • Es una tecnica que ayuda a resolver problemas de conteo para encontrar la solucion y se basa en una multiplicacion sucesiva para determinar laforma en la que puede ocurrir un evento.

    • 2.1.3 Notación Factorial.

      • La notación factorial es un concepto matemático fundamental que se representa con el símbolo "!". Se utiliza para denotar el producto de todos los números enteros positivos desde 1 hasta un número dado.

    • 2.1.4 Permutaciones.

      • Una permutación es una disposición ordenada de elementos distintos donde cada orden es considerado único

    • 2.1.5 Combinaciones

      • Una combinación es una selección de objetos sin considerar el orden en que se eligen.

    • 2.1.6 Diagrama de Árbol.

      • Un Diagrama de Árbol es una representación gráfica que se utiliza para visualizar y calcular las probabilidades de eventos en un proceso de toma de decisiones secuencial o en situaciones donde se deben evaluar diferentes opciones y sus resultados posibles.

    • 2.1.7 Teorema del Binomio

      • Se refiere a una fórmula específica que describe la distribución de probabilidades de una variable aleatoria binomial.

  • 2.2 Teoría elemental de probabilidad.

    • La Teoría elemental de probabilidad es una rama de las matemáticas que estudia los fenómenos aleatorios y sus probabilidades asociadas. Se centra en el análisis de experimentos que pueden tener múltiples resultados posibles, cada uno con una cierta probabilidad de ocurrir. Esta teoría proporciona herramientas y métodos para cuantificar y manejar la incertidumbre en situaciones donde los resultados no son deterministas, sino que siguen una distribución de probabilidades.

    • Tema 3. Variables Aleatorias.

    Subtemas

  • 3.1 Variables aleatorias discretas

      • Una variable aleatoria es discreta cuando sólo puede tomar unos ciertos valores enteros en un número finito de valores o infinito numerable.

    • 3.1.1 Distribución de probabilidad en forma general.

      • cálculo matemático que establece todas las posibilidades que existen de que ocurra un fenómeno en determinadas circunstancias de azar.

    • 3.1.2 Valor esperado

      • El valor esperado suele denominarse media o promedio "a largo plazo". Esto significa que a largo plazo de hacer un experimento una y otra vez, se esperaría este promedio.

    • 3.1.3 Variancia, desviación estándar.

      • La varianza y la desviación estándar son medidas de dispersión o variabilidad, es decir, indican la dispersión o separación de un conjunto de datos.

    • 3.1.4 Función acumulada.

      • La función de distribución acumulativa especifica la probabilidad de que una variable aleatoria sea menor o igual a un valor dado. La función de distribución acumulativa de la variable aleatoria X es la función F(x) = P(X ≤ x).

  • 3.2 Variables aleatorias Continuas:

      • Una variable aleatoria continua es aquella que puede tomar cualquier valor (al menos teóricamente) entre 2 fijados.

    • 3.2.1Distribución de probabilidad en forma general.

      • Una distribución de probabilidad es aquella que permite establecer toda la gama de resultados probables de ocurrir en un experimento determinado

    • 3.2.2 Valor esperado

      • El valor esperado es uno de los conceptos fundamentales en probabilidad, en un sentido más general que la probabilidad misma. El valor esperado de una variable aleatoria de valor real da una medida del centro de la distribución de la variable.

    • 3.2.3 Variancia, desviación estándar.

      • La varianza y la desviación estándar son medidas de dispersión o variabilidad, es decir, indican la dispersión o separación de un conjunto de datos.

    • 3.2.4 Función acumulada.

      • La función de distribución acumulativa es la función que para un valor x, nos da la probabilidad de que la variable aleatoria sea menor o igual que dicho valor x.

    • 3.2.5 Cálculos de probabilidad.

      • El cálculo de probabilidades es el estudio de cómo se determina la posibilidad de ocurrencia de un suceso.

    Tema 4. Distribuciones de Probabilidad.

    Subtemas

  • 4.1 Función de probabilidad.

      • Describe cómo se distribuyen las probabilidades de ocurrencia de los diferentes valores que puede tomar una variable aleatoria discreta. Esta función asigna a cada valor posible de la variable aleatoria la probabilidad de que dicho valor ocurra.

  • 4.2 Distribución binomial.

    • La distribución binomial describe la probabilidad de obtener un número específico de éxitos en un número fijo de ensayos independientes, donde cada ensayo tiene dos resultados posibles (éxito o fracaso) con una probabilidad constante de éxito 𝑝.

  • 4.3 Distribución hipergeométrica

    • La distribución hipergeométrica es un modelo probabilístico que describe la probabilidad de obtener un cierto número de éxitos (por ejemplo, elementos de interés) en una muestra extraída sin reemplazo de una población finita, donde la proporción de éxitos en la población es conocida.

  • 4.4 Distribución de Poisson

    • La distribución de Poisson es un modelo probabilístico que describe el número de eventos que ocurren en un intervalo de tiempo o espacio específico, dado que estos eventos ocurren a una tasa promedio conocida y de manera independiente entre sí.

  • 4.5 Distribución normal.

    • La distribución normal, también conocida como distribución gaussiana, es un modelo matemático que describe la distribución de datos que se encuentran alrededor de un valor medio (media) con una dispersión simétrica alrededor de esta media.

  • 4.6 Distribución T-student.

    • La distribución t de Student, o distribución t de Student, es un modelo probabilístico que se utiliza principalmente en inferencia estadística para estimar parámetros de una población cuando el tamaño muestral es pequeño y/o la desviación estándar de la población es desconocida.

  • 4.7 Distribución Chi cuadrada.

    • La distribución chi-cuadrado es un modelo probabilístico que surge en estadística y se utiliza principalmente en pruebas de hipótesis y en la construcción de intervalos de confianza, especialmente en el contexto de la inferencia sobre la varianza de una población.

  • 4.8 Distribución F

    • La distribución F es un modelo probabilístico que se utiliza principalmente en estadística para comparar varianzas o realizar pruebas de hipótesis sobre la igualdad de varianzas de dos poblaciones normales. También se emplea en análisis de regresión para probar la significancia conjunta de múltiples variables explicativas.

    Tema 5. Regresión lineal.

    Subtemas

    La regresión y correlación son dos conceptos fundamentales en estadística y análisis de datos que se utilizan para medir y estudiar las relaciones entre variables.

    • 5.1.1 Diagrama de dispersión

      • Un diagrama de dispersión es una representación gráfica que muestra la relación entre dos variables cuantitativas. Este tipo de diagrama es fundamental para entender cómo se comportan conjuntamente dos variables y si existe alguna relación entre ellas.

    • 5.1.2 Regresión lineal simple.

      • La regresión lineal simple es un método utilizado para modelar la relación entre una variable dependiente (Y) y una única variable independiente (X).

    • 5.1.3 Correlación

      • La correlación se refiere a la medida estadística que describe la relación entre dos o más variables. Específicamente, se utiliza para determinar si y cómo las variables están relacionadas entre sí, así como la fuerza y la dirección de esta relación.

    • 5.1.4 Determinación y análisis de los coeficientes de correlación y de determinación

      • Se refieren a la evaluación de las relaciones entre variables mediante medidas estadísticas específicas. Estas medidas son fundamentales para entender la fuerza y la dirección de las asociaciones entre variables cuantitativas.

    • 5.1.5 Distribución normal bidimensional

      • Es un concepto estadístico que describe la distribución conjunta de dos variables aleatorias continuas que están normalmente distribuidas. Este tipo de distribución es fundamental en estadística multivariada y se utiliza para modelar situaciones donde las variables están correlacionadas.
